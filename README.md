### Sobre o Projeto

Este projeto foi desenvolvido como parte de um teste prático para avaliar conhecimentos em JavaScript. As aplicações foram passadas por Alex (https://github.com/alexalannunes) para validar as habilidades e implementação. O projeto inclui:

---

### Tecnologias

- [React](https://reactjs.org/) - Biblioteca para construir interfaces de usuário, utilizada para criar a estrutura da aplicação.
- [TypeScript](https://www.typescriptlang.org/) - Superset do JavaScript que adiciona tipos estáticos, utilizado para aumentar a robustez do código.

---

### Como Começar

```bash
# Clone o repositório
$ git clone https://github.com/sillasemanoel/alex-s-challenges

# Navegue até o diretório do aplicativo
$ cd alex-s-challenges

# Instale as dependências
$ npm i

# Para iniciar o aplicativo
$ npm run dev
```
